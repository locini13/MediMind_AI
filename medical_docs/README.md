# MediMind AI — Medical Documents Directory

Place your medical PDF files in this directory.
They will be automatically processed and indexed on server startup.

## Supported documents:
- Medical textbooks
- Disease information documents
- Symptom analysis guides
- Treatment references
- Research papers
- Drug-related documents

## Notes:
- Files are indexed based on MD5 hash, so re-processing only happens when files change.
- Large PDFs may take several minutes to process on first run.
